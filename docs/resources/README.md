# Architecture Diagrams

The architecture diagrams for this project are now maintained in [Mermaid](https://mermaid.js.org/) format.

- `ethio-connect-monorepo.mermaid`: Illustrates the Nx Monorepo Software Design Architecture and the modular CI/CD pipeline.
- `ethio-connect-infrastructure.mermaid`: Depicts the deployment orchestration, including GitHub Actions, GHCR, ArgoCD, Rancher, and the multi-cluster Kubernetes setup.

To view these diagrams, you can use any Mermaid-compatible viewer or editor.
